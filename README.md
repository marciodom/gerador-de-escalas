# Gerador de Escalas

 Aplicação Web desenvolvida como Progressive Web App (PWA) para geração de escalas mensais de trabalho, publicada via GitHub Pages e com suporte a funcionamento offline por meio de Service Worker.
O sistema foi criado para auxiliar na organização de turnos de vigilantes, permitindo geração manual e automática de escalas, com exportação individual em formato JPEG.

* SOBRE A APLICAÇÃO 

 Esta Aplicação Web PWA,funciona gerando escalas manuais e automáticas,com download automático em formato JPEG individual,para a organização do trabalho mensal de vigilantes de uma rede supermercado

🎯 Funcionalidades

• Geração manual de escalas mensais
• Geração automática de escalas
• Organização de turnos por colaborador
• Exportação individual da escala em formato JPEG
• Funcionamento offline
• Instalação como aplicativo no celular (PWA)
